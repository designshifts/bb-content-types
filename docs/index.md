# BB Content Types

BB Content Types lets you create custom post types and taxonomies from a clean admin UI, without writing code.

## What it does

- Define post types with labels, slugs, and visibility
- Create taxonomies and attach them to post types
- Manage rewrite settings and URL structure
- Import/export configurations between sites

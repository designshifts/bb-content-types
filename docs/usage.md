# Usage

## Create a post type

1. Go to **Content Types → Post Types**.
2. Click **Add New**.
3. Set a name, slug, and labels.
4. Choose editor supports (title, editor, featured image, etc.).
5. Save.

## Create a taxonomy

1. Go to **Content Types → Taxonomies**.
2. Click **Add New**.
3. Set a name and slug.
4. Choose related post types.
5. Save.

## Export or import

1. Open **Content Types → Import/Export**.
2. Export your config from one site.
3. Import it into another site to mirror setup.

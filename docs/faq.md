# FAQ

## Who is this for?

Anyone who needs custom post types and taxonomies without writing code.

## Will it change my theme or content?

No. It only registers post types, taxonomies, and rewrite rules.

## Do I need to flush rewrites?

Rewrite changes are flagged and flushed automatically. You can also use the **Flush rewrites** action.

## Can I export configurations?

Yes. Use the Import/Export screen to move settings between sites.
